1.2 ERC721与ERC20的区别

  • 不可替代性：ERC721代币（NFTs）彼此不同，每个都有其独特的属性和价值，而ERC20代币是可互换的，彼此相同。
  • 个体追踪：ERC721允许追踪每个代币的历史和所有者，而ERC20只关注代币的总供应和余额。

2.2 NFT的价值

  • 稀缺性：每个NFT都是独一无二的，这创造了其稀缺性，进而影响其价值。
  • 所有权证明：NFT通过区块链技术记录所有权，确保资产的透明度和不可篡改性。
