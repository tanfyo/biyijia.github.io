1. 直接在币安交易所购买

1.1 创建账户


首先，你需要在币安官网注册一个账户，完成身份验证。

1.2 充值

通过银行转账、信用卡、借记卡或已有的加密货币为账户充值。充值方式取决于你的地理位置和偏好。

1.3 购买BNB

在”交易”页面，选择法币交易或币币交易。法币交易支持直接用法定货币购买BNB，而币币交易则需要你先拥有其他加密货币。

1.4 手续费

在币安，使用BNB支付手续费可享受50%的折扣。标准的法币交易手续费通常为0.1%，币币交易手续费同样为0.1%，但使用BNB支付可降至0.05%。

2. 通过第三方平台购买

2.1 选择平台

例如Coinbase、Kraken或LocalBitcoins等，它们提供了购买加密货币的平台。

2.2 注册与验证

在第三方平台注册并完成身份验证。

2.3 购买BNB

通过这些平台，你可以直接用法定货币购买比特币或以太坊，然后在币安交易所将它们兑换成BNB。

2.4 手续费

第三方平台的手续费因平台而异，通常高于币安。例如，Coinbase的购买手续费可能高达3.99%，而转账到其他交易所可能还需要支付网络费用。

3. 使用P2P交易

3.1 P2P平台

如LocalCryptos或Paxful，允许用户直接与个人卖家交易。

3.2 交易流程

发布购买请求，选择合适的卖家，完成支付并确认交易。

3.3 手续费

P2P平台通常收取交易手续费，一般在1%左右，卖家可能还会设定额外费用。

4. 通过ATM购买

4.1 寻找加密货币ATM

在你的城市找到支持BNB的加密货币自动取款机。
