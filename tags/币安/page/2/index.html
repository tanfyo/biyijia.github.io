网格交易是一种在预设价格区间内，设定多个买入和卖出点的策略。通过在价格波动中不断买入和卖出，实现低买高卖，赚取差价。

  • 自动化交易：无需时刻关注市场，设置好参数后，系统自动执行交易。
  • 波动性利用：适合震荡市场，通过价格波动获取利润。

  • 网格数量：决定在价格区间内设置多少个买入和卖出点。
  • 网格间距：每个网格的价格差，决定买入和卖出的频率。
  • 启动资金：分配给网格交易的初始资金。
  • 止盈止损：设置盈利和亏损的触发点，以保护本金。
