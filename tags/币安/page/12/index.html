2. 格式调整

  • 压缩文件：将文件大小压缩到允许的范围内。

3. 核对信息

  • 仔细核对：确保填写的个人信息与身份证件完全一致。
  • 更新地址：如果地址有变，及时更新个人信息。

4. 完成所有步骤

  • 上传双面：如果需要，确保上传身份证件的正反两面。
  • 重新进行面部识别：按照指引完成面部识别过程。
