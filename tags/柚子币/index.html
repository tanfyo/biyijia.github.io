1.2 EOS的特点

  • 免费交易：用户无需支付交易费用，而是通过持有EOS代币来获得网络资源。
  • 权限和治理：EOS采用基于角色的权限管理，以及社区驱动的治理模式。

2. 准备工作

选择一个交易所，完成注册并验证账户。

4. 充值法币

4.1 银行转账

大多数交易所支持银行转账，但可能需要几天时间到账。

4.2 信用卡/借记卡

在交易所完成购买后，将EOS提至你的MetaMask或其他安全的钱包。
